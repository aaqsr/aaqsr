## Development

Install dependencies:
```bash
bun install
```

Start development server:
```bash
bun run dev
```

Build for production:
```bash
bun run build
```

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the main branch.

To deploy manually:
```bash
bun run deploy
```
